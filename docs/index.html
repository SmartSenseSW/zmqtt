ZMQTT Debian Repository

This is the official package repository for the ZMQTT Service.

Installation

Run the following commands on your Raspberry Pi:

# 1. Add the repository (Trusting unsigned)
echo "deb [trusted=yes] https://smartsensesw.github.io/zmqtt stable main" | sudo tee /etc/apt/sources.list.d/zmqtt.list

# 2. Update and Install
sudo apt-get update
sudo apt-get install zmqtt-service
